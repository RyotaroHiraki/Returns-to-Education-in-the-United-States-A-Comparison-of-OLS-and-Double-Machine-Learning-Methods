# Data Access Instructions

The raw dataset used in this project is not publicly distributed due to licensing restrictions.

To obtain the data, please visit the official source:

- [Official CPS 2024 ASES Data Source](https://www.census.gov/data/datasets/time-series/demo/cps/cps-asec.2024.html#list-tab-165711867)

And download: *CSV Data File (includes Replicate Weights)*

After downloading, place the data files in this directory.

Make sure file names match those expected in the code.
